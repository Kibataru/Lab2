# Лабораторная 2

## Описание
Был сделан пайплайн для извлечения данных из текста

**Выбранная задачка:** Извлечение характеристик товаров из текстовых описаний

**Работа программы:**
1. Чтение входные неструктурированные данные из файла `product.csv`
2. С помощью `openai` делаем запрос к модели через API MiMo. Взята  модель `mimo-v2.5-pro`
3. В коде применен рrompt, он задает формат вывода. Описание товара передается в специальных тегах-кавычках
4. Извлекаются три параметра: **Бренд** (`brand`), **Категорию** (`category`) и **Цену** (`price`). Отсутствие данных -  `null`
5. Итоги парсятся в словарь и сохраняются в `output.json`

---

## Пример данных

**product.csv**
```csv
id,description
1,"Смартфон Apple iPhone 15 Pro Max 256GB Natural Titanium. Отличная камера и батарея. Текущая цена 145 000 рублей."
```
## Пример выходных данных

**output.json**
```json
[
    {
        "brand": "Apple",
        "category": "Смартфон",
        "price": "145 000 рублей",
        "id": "1"
    }
]
```
## Инструкция по запуску

1. **Склонируйте репозиторий**
2. pip install -r requirements.txt
3. ** В .env впишите ключ и адрес**
